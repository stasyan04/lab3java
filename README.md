### 1. Клонування репозиторію
git clone https://github.com/stasyan04/lab3java

cd lab3java

javac com/stanislav/labwork3/SeaBoat.java com/stanislav/labwork3/SeaBoatDemo.java

java com.stanislav.labwork3.SeaBoatDemo
